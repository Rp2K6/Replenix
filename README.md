# ✈️ Replenix – Smart Inventory Replenishment System

Replenix is an intelligent inventory system that predicts stockouts and automates replenishment in real time.

## 🚀 Features

* 📦 Real-time inventory tracking
* ⚠️ Low stock alerts
* 🔮 Prediction of stock depletion
* 🤖 Automated reorder trigger
* 🧠 AI-based explanation system
* 🔄 Live simulation dashboard

## 🛠️ Tech Stack

* Backend: FastAPI
* Frontend: Streamlit
* Database: SQLite
* Language: Python

## ▶️ How to Run

### Backend

```bash
cd backend
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend_project
streamlit run app.py
```

## 🌐 Architecture

Frontend → Backend → Database
